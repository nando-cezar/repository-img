1. No terminal digite os comandos
a) id
b) who
c) whoami

2. Houve alguma diferença nos resultados apresentados? Se sim, quais foram? Se não houve diferença,
o que deveria, justifique e apresente a diferença entre os comandos.
	O comando id no Linux é usado para descobrir nomes de usuários e grupos e IDs numéricos (UID ou ID de grupo) 
	do usuário atual ou de qualquer outro usuário no servidor.

	O comando who é uma ferramenta de impressão de informações sobre os usuários que estão logados no momento.

	O comando whoami permite que os usuários do Linux vejam o usuário conectado no momento.
